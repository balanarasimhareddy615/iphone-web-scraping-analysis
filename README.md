# iPhone Web Scraping & Analysis

# Overview
This project demonstrates web scraping and data analysis using Python.
Product data such as price, storage, ratings, and specifications were collected
and analyzed to identify trends and insights.

# Tools & Libraries
- Python
- BeautifulSoup
- Requests
- Pandas
- Matplotlib

# Key Steps
1. Web scraping product details
2. Data cleaning and preprocessing
3. Exploratory data analysis
4. Visualization of price and storage trends

# Key Insights
- Storage size significantly affects product price
- All variants maintain high customer ratings
- Specifications are consistent across models

# Future Improvements
